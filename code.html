CMB lensing power spectra computed from matter power spectra measured from various suites of hydrodynamical simulations (used in Chung et al. 2020) are hosted at this Github repository.

Power spectrum and bispectrum measurements from various suites of hydrodynamical simulations (used in Foreman et al. 2020) are hosted at this Github repository.


Code

I have written several research codes that are publicly available, including:

  • CMBnoise: a simple Python module to compute CMB noise power spectra for individual frequency bands, in units of uK^2 or Jy^2 sr^-1. (Used in Breysse et al. 2021.)

  • bskit: a Python code (built upon nbodykit) for measuring bispectra from cosmological simulation snapshots. (Used in Foreman et al. 2020.)

  • StringFast: a fast Foretran 90 code to compute CMB power spectra induced by a network of cosmic strings (Used in Foreman et al. 2011.)

I am also a regular contributor to the radiocosmology suite of codes for simulation and analysis of data from transit radio interferometers, and have contributed to codes for perturbation theory calculations, which can be found at the EFTofLSS code repository.
