Code

I have written public codes for measuring density bispectra from cosmological simulations and computing the effect of cosmic strings on CMB temperature and polarization power spectra; both can be found at my GitHub profile. I have also contributed to codes for perturbation theory calculations, which can be found at the EFTofLSS code repository.
